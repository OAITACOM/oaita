## Overview
OAITA delivers a fast, private, and fully offline AI chat experience through a modern browser interface.  
Launch once, and your default browser opens to a responsive chat window powered entirely by your local hardware.  
No internet connection is required, and no data ever leaves your machine.

## How It Works
1. **Launch**: Open `oaita.exe`. 
2. **Choose a model**: Use the included model, or downloaded more models from the website.
3. **Start server**: OAITA starts a lightweight local web server and opens the chat interface in your browser.  
4. **Connect from other devices**: If LAN mode is enabled, open the displayed local IP on any device connected to your network.  
5. **Chat**: Type messages in the browser interface. The AI model processes them locally and returns responses.  
   *(Note: the first message may take longer as the model warms up.)*
